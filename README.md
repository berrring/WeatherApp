WeatherMe — простое и стильное веб-приложение для просмотра погоды, вдохновлённое дизайном Apple Weather.
Особенности

Поиск любого города
Три вкладки: Today (почасовой прогноз от текущего/ближайшего часа), Tomorrow (полные 24 часа), Weekly Forecast (7 дней)
Переключение °C / °F с автоматическим пересчётом температуры, ветра и видимости
Часы показывают локальное время выбранного города
Красивый тёмный интерфейс с градиентами и скроллящейся каруселью прогноза
Полностью на vanilla HTML + CSS + JavaScript
Данные берутся с бесплатного Open-Meteo API (без ключей)
